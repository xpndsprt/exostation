Notes

New sprites are 32×32 per tile (multi-tile items expand the grid). Loading an older 16px built-in keeps its native size. Each sprite holds named states (creatures: idle/walk/dead; modules: enabled/disabled). Left-drag paints, right-drag erases. JSON saves every state; PNG saves the current state; PNG sheet saves all states in a row.
